# Project Guidelines

## Commands
- `make build` - Build the project
- `make test` - Run tests

## Technologies
- Go Language
- Docker
- GlusterFS

## Supported Architectures
- x86_64
- arm64

## Development Workflow
- Use `make build` to build the project
- Use `make test` to run tests
- Ensure the code follows Go language best practices
- Use Docker for containerization
- Test on both x86_64 and arm64 architectures
